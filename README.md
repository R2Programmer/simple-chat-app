### Installation

Start with cloning repo, then go to:

```bash
cd <project>
```

To install and set up the library, run:
```bash
$ npm install
```

Start the app on the client-frontend.
```bash
npm start
```
Start the app on the socket.
```bash
npm start
```

Start the app on the server-side.
```bash
npm run dev
```
